# Educational Website (HTML Only)

## Requirements Definition

### Aim
The aim of this project is to create a simple educational website using **HTML only**.  
The website will display learning content clearly in a web browser and allow users to move through information easily.


## Functional Requirements

### Website Structure
- The website must be built using HTML.
- It must use headings, paragraphs, and sections so content is easy to read.

### Navigation
- The website must allow users to move between pages or sections using links.
- All links must work correctly and not lead to broken pages.

### Content Display
- The website must clearly show instructional content such as text, lists, or images.
- All content must load properly when the page opens.

### User Interaction
- Users must be able to interact with the website by clicking links or buttons to move between sections.

### Data Organisation
- Information must be organised using HTML elements such as lists, tables, or sections.


## Non-Functional Requirements

### Performance
- The website should load quickly in a web browser.

### Usability
- Users should be able to find all learning content within **three clicks** from the home page.

### Accessibility
- The website must be easy to read with clear text, headings, and spacing.

### Compatibility
- The website must work in common web browsers such as Chrome, Edge, and Safari.


## Constraints

### Technology
- Only **HTML** may be used.
- No CSS, JavaScript, or backend code is allowed.

### Resources
- Only free tools may be used to create the website.

### Time
- The project must be completed by **Term 2, Week 2**.


## Acceptance Criteria
All buttons and links work correctly.
- The website responds properly to user input.
- Users can navigate between sections without errors.
- The program includes at least one data structure and functions.
- The website runs without major bugs.

### Navigation
- All links work correctly and allow users to move between pages.

### Usability
- All learning content can be accessed in three clicks or fewer.

### Structure
