## One-Page Report

### Purpose
The purpose of this project is to create a simple educational website that helps users learn basic Software Engineering concepts. The website is designed to display information clearly and allow users to move through content easily using a web browser. It aims to support beginner learners by providing structured and easy-to-understand material.



### Design Decisions
Several design decisions were made to keep the website simple and effective:

- **HTML Only**: The website was built using only HTML to meet project constraints and keep the design simple.
- **Clear Structure**: Headings, paragraphs, and sections were used to organise content and improve readability.
- **Simple Navigation**: Links were used to allow users to move between pages easily.
- **Minimal Layout**: The design avoids clutter so users can focus on learning content.
- **Three-Click Rule**: Content is organised so users can find information quickly within three clicks.

These decisions ensure the website is easy to use, especially for beginners.


### Functionality
The website provides the following features:

- Users can navigate between different pages using links.
- Educational content is displayed clearly using text, lists, and headings.
- Pages load quickly without delays.
- All links connect correctly to other sections of the website.
- The layout remains consistent across all pages.

The system focuses on delivering content in a simple and reliable way without advanced features, due to the HTML-only limitation.


### Summary
Overall, the website meets its goal of providing a simple and accessible learning tool. It is easy to navigate, clearly structured, and suitable for beginner users.
